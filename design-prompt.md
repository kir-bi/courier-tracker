# Промпты для Claude Design — Courier Tracker

---

## 1. Продающий сайт (лендинг)

Создай мокап продающего лендинга для **Courier Tracker** — B2B SaaS для ресторанов с собственной курьерской службой.

### Стиль

Вдохновлён сайтом PAJ GPS (paj-gps.es):
- Тёмно-синий / тёмно-графитовый фон на hero-секции (~#0d1b2a)
- Яркий оранжевый акцент (#ff6b1a) для CTA, выделений и иконок
- Крупная жирная белая типографика в заголовках
- Мокапы телефонов и планшетов с картой прямо в hero
- Чередование тёмных и светлых секций
- Чистый, современный B2B-стиль

### Структура страницы

**Hero:**
- Заголовок: «Знай где твой курьер. Всегда.»
- Подзаголовок: для ресторанов с собственной доставкой — реальный ETA вместо "примерно через 40 минут"
- Кнопка: «Попробовать бесплатно»
- Мокап: 2 телефона — один показывает карту менеджера с метками курьеров, второй — экран курьера "начать смену"

**Боль → Решение:**
- 3 карточки: нет видимости → карта реального времени / клиенты звонят → точный ETA / Excel и звонки → всё в одном экране

**Как это работает:**
- 3 шага с иконками: Курьер открывает приложение → Менеджер видит карту → Клиент получает ссылку

**Для кого:**
- Рестораны и фудкорты с собственной доставкой (не Яндекс/Glovo)
- Акцент на малый-средний бизнес: 2–20 курьеров

**Скриншоты продукта:**
- Дашборд менеджера (веб, полный экран)
- Мобильное приложение курьера
- Страница live-трекинга для клиента

**Тарифы (заглушка):**
- 2–3 тарифных плана с ценой за курьера/месяц
- Выделить средний план как "Популярный"

**CTA внизу:**
- «Подключить ресторан» — тёмный блок с оранжевой кнопкой

## 2. Мобильное приложение + Админка

Создай мокап трёх экранов для **Courier Tracker**.

### Стиль

- Тёмная тема (dark UI): фон #0f1923, карточки #1c2a38
- Оранжевый акцент #ff6b1a для статусов, кнопок и активных состояний
- Карта с тёмным стилем (Mapbox dark / Google Maps night mode)
- Минималистично — приложение для работы, не для развлечения

### Экран 1 — Дашборд менеджера (мобайл/планшет)

- Карта на весь экран с метками курьеров (цветные пины по статусу: в пути / ждёт / задержка)
- Снизу — выезжающая панель со списком активных заказов: адрес, курьер, ETA
- Фильтр по курьерам в шапке
- Метка курьера при тапе показывает мини-карточку: имя, текущий заказ, ETA

### Экран 2 — Приложение курьера (мобайл)

- Главный экран: большая кнопка «Начать смену» (неактивное состояние)
- После нажатия: список заказов на день — адрес, имя клиента, временной слот
- Минимум элементов — курьер за рулём, интерфейс должен читаться с одного взгляда

### Экран 3 — Live-трекинг для клиента (веб, мобайл-версия)

- Простая страница: «Ваш заказ из [Ресторан] уже в пути»
- Карта с меткой курьера и адресом доставки
- ETA крупно: «Прибудет через 12 минут»
- Лого ресторана в шапке (white-label)
